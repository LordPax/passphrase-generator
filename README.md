# Passphrase generator

## Description
Petit script permettant de générer des passphrases aléatoirement

## Usage
```
Usage : $0 [options]

Option :
-h or --help ...................... This help
-v or --version ................... Show version
-n <nbWord> ....................... Number of word 
-l <length> ....................... Max length of word
-w <file> ......................... Wordlist
-s ................................ Statistiques (soon)
```
