# Passphrase generator

## Description
Petit script permettant de générer des passphrases aléatoirement

## Usage
```
Usage : $0 [options]

Options :
-h or --help ...................... This help
-n <nbWord> ....................... Number of word (soon)
-l <length> ....................... Max length of word
-w <file> ......................... Wordlist
-s ................................ Statistiques (soon)
```
